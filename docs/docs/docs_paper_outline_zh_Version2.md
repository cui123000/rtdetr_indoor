# 室内目标检测中的高效注意力增强 MobileNetV4（短文大纲）

## 1. 题目与作者
- 题目：面向室内目标检测的高效注意力增强方法
- 作者、单位、邮箱

## 2. 摘要（150–200字）
背景：室内场景目标密集、遮挡多，且移动端部署对延迟与算力敏感。
方法：在 RT-DETR 框架中，将轻量级 SEA 注意力嵌入 MobileNetV4 主干，形成 MNV4-SEA。
结果：在 HomeObjects-3K 上，相比 MNV4 基线，mAP50 提升约 +14.4%，参数量约 29.06M；在 COCO-Indoor 子集上的验证显示良好跨数据集泛化；同时在参数、FLOPs、延迟方面形成更优的精度–效率折中。
贡献：给出一套适用于移动友好型主干的注意力插入范式与系统消融；代码、配置与数据子集生成脚本开源。展望：融合 CLIP 的指令驱动检测。

## 3. 引言（约0.5页）
- 动机：室内检测的挑战（小目标、遮挡、杂乱）、移动/边缘端约束。
- 问题：传统融合颈部（如 ASFF/BiFPN）在 RT-DETR 上破坏原有平衡；需要更轻、更稳的增强策略。
- 贡献概述：
  1) 提出 MobileNetV4 中的 SEA 轻量注意力插入方案（插入位置与超参可复用）。
  2) 系统消融，区分结构增益与训练技巧增益。
  3) 完整效率评测（参数/FLOPs/延迟/显存）与 COCO-Indoor 跨数据集验证。
  4) 可复现资产：代码、配置、COCO-Indoor 生成脚本与训练命令。

## 4. 相关工作（约0.3页）
